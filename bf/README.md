# BrainFuck Modules

This directory holds the BF-side implementation.

- `lib_hex_decode.bf` - hex pair to byte
- `lib_big_endian.bf` - 2-byte and 4-byte big-endian helpers
- `phase1_header_inspector.bf` - request header and consume the reply
- `phase2_header_parser.bf` - validate SQLite header and page size
- `phase3_page1_parser.bf` - parse page 1
- `phase4_schema_walk.bf` - walk `sqlite_schema`
- `phase5_table_scan.bf` - scan the demo `users` table
- `phase6_insert.bf` - controlled insert demo
- `phase7_update.bf` - controlled update demo
- `phase8_delete.bf` - controlled delete demo
- `phase9_select_users_name.bf` - project one column from the demo `users` table
- `phase9_select_users_name_sex.bf` - project two columns from the demo `users` table

Most later phase `.bf` files are generated by the scripts in `scripts/`.
